Poner imagen en SD desde Windows

Poner la imagen de la distribución elegida es muy fácil desde Windows ya que se hace mediante el programa Win32diskimager que es muy fácil de usar y gratuito.
Cuando se tenga instalado el programa, insertamos la tarjeta SD vacía en el equipo e iniciamos el programa como administrador, es muy importante iniciarlo como administrador ya que sino no funcionará bien. Una vez iniciado se abrirá el programa donde indica el dispositivo conectado, en el caso que haya conectado más tarjetas SD habrá que elegir la letra correcta porque se formateará.
En Image File abrimos la imagen a instalar desde la carpeta y le damos a Write. Recordar que cuando se descarga las distribuciones se tienen que extraer para obtener la imagen.
Nos saldrá un mensaje de advertencia de que se puede corromper le damos a Yes.
Cuando se acabe de escribir la imagen saldrá un aviso conforme se a escrito le damos al OK y se cierra el programa y se extrae la tarjeta SD.

Poner imagen en SD desde Linux

Poner la imagen desde Linux mediante la terminal a mi modo de ver se aprende más que no de manera gráfica de modo que voy a explicar los pasos necesarios.
Primero se inserta la tarjeta SD en nuestro Linux hay que fijarse en el nombre que le da nuestro Linux suele ser /dev/sdc o cualquier variante. Si el S.O monta automáticamente la tarjeta SD, se tiene que desmontar para poder trabjar con ella. de modo que se desmonta poniendo en la terminal lo siguiente:

# umount /dev/sd**

Una vez desmontada comprobamos que nuestra tarjeta SD no posee una tabla de particiones con el comando fdisk

Formatear SD

En el caso de que la tarjeta SD no este vacía habrá que formatearla antes, aunque el mismo programa Win32diskimager ya la formatea, si no esta vacía no funciona bien.
Existen varios programas para formatear una SD pero desde Windows por desgracia si la tarjeta SD tiene alguna distribución de Linux dan problemas y no llegan a formatearse bien. Tales como SD Formater. Hay alternativas a ese programa como HP-USB-Disk-Storage-Format-Tool, pero yo el que recomiendo es usar desde el Gparted de Linux en el que si reconoce las particiones de Linux, el cuál en esta página indica como se pone la imagen en un USB y poder utilizar dicho programa.
Una vez estamos dentro y hemos seleccionado la tarjeta SD tendrá de nombre parecido a mmcblk0 si tiene un Linux dentro. Se selecciona la partición que este creada y le damos a Delete.
Después se selecciona todo el tamaño y se formatea al sistema de archivos Fat32
