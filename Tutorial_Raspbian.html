Instalación de Raspbian

Para empezar se deberá tener la imagen de Raspbian en la tarjeta SD, tal y como se indica en este enlace, una vez ya la tengamos encendemos nuestra Raspberry Pi y cargará el sistema.

Expansión del sistema de archivos

Cuando acaba de cargar el sistema saldrá la herramienta llamada raspi-config donde se procederá a configurar e instalar Raspbian.
Manos a la obra en la primera opción es para expandir el sistema de archivos para que ocupe toda la tarjeta de modo que es lo primero que se hace.
Después de seleccionar la primera opción saldrá una ventana conforme se realizará el proceso al reiniciar de modo que se selecciona OK

Cambio de contraseña

Debido al usuario creado por defecto Pi y su contraseña raspberry hay que cambiar la contraseña de dicho usuario para mayor seguridad. Para ello en la pestaña principal de raspi-config en la segunda opción se puede cambiar la contraseña, le damos y se abrirá una ventana que permitirá cambiar la contraseña, al escribir la nueva contraseña no se visualizará de modo que cuidado cuando se escribe la nueva contraseña.
Una vez cambiada saldrá otro aviso conforme se ha cambiado la contraseñ se selecciona OK y se volverá a la pestaña principal.

Escritorio

En la distribución Raspbian se puede elegir si instalar o no la interfaz gráfica para que se ejecute autmáticamente al encender el sistema. El tipo de escritorio es LXDE se trata de un escritorio bastante reducido de peso. Se selecciona la tercera opción.
Una vez dado saldrá una ventana donde se podrá elegir si se quiere instalar o no, por defecto no se instala de modo que bajo mi punto de vista no le veo sentido marcar la opció NO.

Configuración regional

Dentro de Internationalisation Options hay un menú en el cuál se puede configurar el lenguaje, zona horaria y el teclado. Así que seleccionamos esa opción.
Cuando se haya seleccionado se verá el menú, de modo que se selecciona la primera opción Chage Locale para cambiar el idioma y carácteres.
Una vez dentro se selecciona el idioma con la barra de espacio del teclado para marcar las opciones, y después a OK en este caso selecciono es_ES.UTF-8 UTF-8 ya que mi idioma es el español y vivo en españa con el juego de carácteres UTF-8.
Ahora saldrá otra ventana donde hay que configurar el conjunto de carácteres que usará por defecto que en este caso es el mismo de antes. Después hay que darle al OK.
De nuevo regresará al menú regional, ahora toca seleccionar la segunda opción para cambiar la zona horaria.
Se selecciona el Continente al que se pertenece.
Luego toca seleccionar el país o ciudad al que se pertenece, (es posible que no salga vuestra ciudad ni el país, pero saldrá la capital del país).
De vuelta otra vez en el menú regional seleccionamos la última opción para configurar el teclado.
Ahora viene cuando se tiene que seleccionar el modelo del teclado conectado, si no se encuentra en la lista hay que seleccionar Generic y el número de teclas que tiene el teclado.
Después toca seleccionar el idioma de nuestro teclado si es español como es en este caso, se elige la opció Other.
Se selecciona el país de origen del teclado.
Una vez hecho toca seleccionar la disposición de las teclas, si no se esta seguro seleccionar la primera opción.
Ahora toca elegir que función servirá como la tecla ALT GR, si ya tenemos una tecla específica, se elige la primera opción.
Luego toca elegir que tecla funcionará como la tecla Compose o Multi-key si no se esta seguro de lo que se hace marcar la primera opció.
Por último permite habilitar una combinación de teclas para acabar con el servidor X Si no sabéis lo que hacéis marcar No.

Habilitar cámara

De vuelta en el menú principal la opción número 5 permite habilitar la cámara si se tuviera se selecciona esa opción.
Saldrá una ventana preguntando si queremos habilitarla o no, si no se tiene una cámara no es necesario entrar en esta opción ya que por defecto no esta habilitada.

Opciones avanzadas

De vuelta en el menú principal pasamos a la opción número 8, me salto la opción 6 ya que es para permitir añadir la Raspberry Pi en el mapa del mundo para usarla como estadística, y la opción 7 que es para hacerle overclock a la Raspberry Pi y no es el caso (Recordar que como máximo se pude overclockear hasta 1 GHz sin perder la garantía).La opción 9 es información sobre la herramienta raspi-config.
Dicha opción 8 se encuentra un menú donde se puede configurar el servidor SSH, cambiar el nombre de la máquina entre otras cosas. Es importante habilitar el servidor SSH para poder controlar la Raspberry Pi remotamente así que seleccionamos esa opción.
Dentro del menú se encuentra las opciones overscan que sirve para configurar la pantalla por si hubieran barras negras, Hostname que sirve para cambiar el nombre de la máquina, por defecto es Raspberry Pi, recomiendo cambiarla aunque no es obligatorio.Memory Split que sirve para cambiar el tamaño de memoria disponible de la GPU, SSH que es para habilitar el servidor y así poder controlar la Raspberry Pi, y por último Update que sirve para actualizar el S.O.
Explicado las opciones seleccionamos la del SSH que es la más importante.
Una vez dentro preguntará si deseamos habilitar el servidor, seleccionamos Enable.
Después saldrá otra ventana conforme se ha habilitado el servidor.

Reiniciar

Cuando ya se acabe de configurar todo es hora de reiniciar para que se apliquen los cambios realizados, para ello en el menú principal seleccionamos Finish.
Saldrá una ventana preguntando si queremos reiniciar seleccionamos Yes para que se reincie.
